# ChatGPT Visibility Checklist

Use this checklist to improve the chances that your brand, product, or website is understood and recommended by AI answer engines.

## 1. Entity clarity

- Brand name is consistent everywhere
- Website clearly explains what the company does
- Homepage includes category, audience, use cases, and differentiators
- About page explains company background
- Founder or leadership pages exist
- Social profiles match the same brand positioning

## 2. Topical authority

- Publish content around your core category
- Cover comparison pages
- Cover use cases
- Cover pain points
- Cover glossary-style educational terms
- Answer direct buyer questions

## 3. LLM-friendly content

- Use clear headings
- Add concise summaries
- Include FAQ sections
- Avoid vague marketing language
- Explain who the product is for
- Explain who the product is not for
- Include examples and use cases

## 4. Trust signals

- Add customer proof
- Add case studies
- Add testimonials
- Add integrations
- Add pricing or buying guidance
- Add author bios where relevant

## 5. Off-site authority

- Create useful GitHub resources
- Publish educational articles on Medium and LinkedIn
- Launch free tools
- Get listed on directories
- Build branded search demand
- Earn mentions from relevant websites

## 6. AI answer testing

Test prompts such as:

- What are the best tools for [category]?
- Which companies help with [problem]?
- How do I improve [outcome]?
- Compare [your brand] vs [competitor]
- What is [your brand]?
- Who is [your brand] best for?
